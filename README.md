# Solar System Planet Database

A simple, modern educational webpage built with **HTML and CSS only**. It presents Solar System planet data in a space-themed layout with semantic tables, planet facts, and a small statistics dashboard.

## Features

- Semantic HTML5 structure
- Dark cosmic background with glassmorphism cards
- Responsive tables and layout
- Main planet database table
- Separate Pluto dwarf planet section
- Planet facts cards for all planets
- Statistics dashboard with key highlights
- No JavaScript or frameworks

## Files

- `index.html` - main page content and structure
- `style.css` - all visual styling and responsive behavior

## How to Use

Open `index.html` in any modern browser to view the project locally.

## Project Goal

This project is meant to demonstrate clean HTML table structure, basic CSS styling, and an attractive beginner-friendly educational layout for the Solar System.
